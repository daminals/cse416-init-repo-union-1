## Bootstrap by team: Portuguese Man O' War
Alexander Snit, Daniel Kogan, Dylan Scott, Gretta Halollari

### Running the code

To run mock market server:
```bash
cargo run --bin market-server
```

To run consumer:
```bash
cargo run --bin peer-node
```

To run producer:
```bash
cargo run --bin peer-node -- -p  
```
